# OneBitBot

Aplicação do tipo chatbot com funções de FAQ que permite aos usuários armazenar perguntas e respostas (listar, remover e pesquisar) com integração com o telegram

# Stacks do projeto

Sinatra
Postgresql
Dialogflow
Docker
Docker Compose
Telegram
